```r

devtools::install_github("BlakeRMills/MetBrewer") 
```

![Ex1](https://github.com/BlakeRMills/MetBrewer/blob/main/Works%20and%20Palettes/Examples/Example%201.png)

```r
met.brewer("Manet", 5)
```
![Ex2](https://github.com/BlakeRMills/MetBrewer/blob/main/Works%20and%20Palettes/Examples/Example%202.png)


```r

```
